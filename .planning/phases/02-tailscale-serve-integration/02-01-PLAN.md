    **Types:**
    ```typescript
    export interface TailscaleInfo {
      hostname: string;   // Self.HostName
      tailnet: string;    // MagicDNSSuffix
      fqdn: string;       // Self.DNSName with trailing dot stripped
      url: string;        // https://<fqdn>
    }

    export type TailscaleStatusResult =
      | { ok: true; info: TailscaleInfo }
      | { ok: false; reason: 'not-connected' | 'invalid-status' | 'cli-error'; stderr?: string }

    export class TailscaleServeError extends Error {
      constructor(
        message: string,
        public readonly exitCode: number | null,
        public readonly stderr: string,
      ) {
        super(message);
        this.name = 'TailscaleServeError';
      }
    }
    ```

    **Pure functions (no I/O):**
    - `parseTailscaleStatus(statusJson: unknown): TailscaleInfo | null` — validates that `statusJson` is an object with `Self.DNSName` (string) and `MagicDNSSuffix` (string) before constructing result. Strips trailing dot from DNSName via `.replace(/\.$/, "")`. Returns null if required fields missing.
    - `buildServeCommand(localPort: number): string[]` — returns `["serve", "--bg", "--https", "443", "https+insecure://127.0.0.1:${localPort}"]`
    - `buildServeResetCommand(): string[]` — returns `["serve", "reset"]`
    - `getInstallCommand(platform: NodeJS.Platform): string` — "darwin" returns `"brew install tailscale"`, "win32" returns `"winget install Tailscale.Tailscale"`, all others return `"curl -fsSL https://tailscale.com/install.sh | sh"`

    **I/O functions (use child_process):**
    - `isTailscaleInstalled(): boolean` — uses `spawnSync("tailscale", ["version"], { stdio: 'pipe' })`, returns `result.status === 0`. Catches exceptions and returns false.
    - `getTailscaleStatus(): TailscaleStatusResult` — uses `spawnSync("tailscale", ["status", "--json"], { stdio: 'pipe' })`. On non-zero exit returns `{ ok: false, reason: 'not-connected' }`. On parse failure returns `{ ok: false, reason: 'invalid-status' }`. On exception returns `{ ok: false, reason: 'cli-error', stderr: error.message }`. On success passes parsed JSON to `parseTailscaleStatus`; if null returns `{ ok: false, reason: 'invalid-status' }`, else returns `{ ok: true, info }`.
    - `startTailscaleServe(localPort: number): Promise<void>` — uses promisified `execFile("tailscale", buildServeCommand(localPort))`. On error, throws `new TailscaleServeError(message, exitCode, stderr)` where `exitCode` comes from the error object and `stderr` from `error.stderr` or error message.
    - `stopTailscaleServe(options?: { strict?: boolean }): Promise<void>` — uses promisified `execFile("tailscale", buildServeResetCommand())`. If `options?.strict` is true, throws `TailscaleServeError` on failure. Otherwise swallows errors silently.
    - `stopTailscaleServeSync(): void` — uses `spawnSync("tailscale", buildServeResetCommand(), { stdio: 'pipe' })` wrapped in try/catch. Never throws. This is the last-ditch cleanup for `process.on('exit')` handlers where async code cannot run.

    Use `import { spawnSync, execFile } from "node:child_process"` and `import { promisify } from "node:util"`.
    Keep `isTailscaleInstalled` and `getTailscaleStatus` synchronous (spawnSync) since they are preflight checks that run serially before anything is spawned.
    Keep `startTailscaleServe` and `stopTailscaleServe` async (execFile) since they run during the server lifecycle.
    `stopTailscaleServeSync` is sync (spawnSync) specifically for the `process.on('exit')` handler.
