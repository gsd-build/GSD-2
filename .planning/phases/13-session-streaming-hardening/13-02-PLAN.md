<objective>
Harden the gsd process lifecycle: add graceful interrupt, crash recovery event emission, and a killAll safety net for orphan prevention.

Purpose: Without interrupt(), the Escape key in auto mode has no mechanism. Without crash event emission, a crashed gsd process silently leaves the UI stuck in "processing" state. Without killAll(), orphaned gsd processes survive app restarts.

Output:
- ClaudeProcessManager.interrupt() — sends SIGINT, no-op if no active process
- crash recovery: when process exits non-zero after having produced output (chunkCount > 0), emit { type: "process_crashed", exitCode, stderr } to handlers
- SessionManager.killAll() — iterates all sessions and calls processManager.kill()
</objective>
